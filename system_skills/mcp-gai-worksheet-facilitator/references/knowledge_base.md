🔵 2. CORE AI CONCEPTS

2.1 Generative AI (GAI)

GAI is defined by three core capabilities:
	•	Language
	•	Understanding
	•	Reasoning

Unlike traditional systems:
	•	Not just right/wrong answers
	•	Provides reasoning and process

⸻

2.2 Large Language Models (LLM)
	•	Built on massive language data
	•	Not a search engine
	•	Predicts relationships between words

Strengths:
	•	Communication
	•	Content generation
	•	Coding support

Limitations:
	•	Precision (e.g. math, exact logic)

⸻
