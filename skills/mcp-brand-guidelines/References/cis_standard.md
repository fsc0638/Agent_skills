# R&D CIS Standard (研發組品牌標準)

## Color Palette Reference Table

| Category | Item | HEX Code | Design Intent |
| :-- | :-- | :-- | :-- |
| **Core** | Brand Blue | `#003366` | Stability, Professionalism, Tech Expertise |
| **Core** | Brand Orange | `#FF6600` | Automation Energy, Action, Important Alerts |
| **Hover** | Blue Hover | `#002244` | Interaction Feedback |
| **Hover** | Orange Hover | `#E65C00` | Interaction Feedback |
| **Layout** | Primary BG | `#FFFFFF` | Clean Dashboard Feel |
| **Layout** | Section BG | `#F9FAFB` | UI Hierarchy differentiation |
| **Typography**| Main Text | `#111827` | High contrast accessibility |
| **Typography**| Sub Text | `#4B5563` | Secondary information |

## Design UI Pattern Rules

1. **Buttons**: 
   - Primary action (Submit, Analyze, Upload) should use **Brand Orange**.
   - Navigational or secondary actions should use **Brand Blue** or subtle variants.
2. **Sidebar/Heade**r:
   - Use **Brand Blue** as the anchoring element.
3. **Cards**:
   - Use `#FFFFFF` with a subtle border `#E5E7EB` and perhaps a shadow.
4. **Agent Thought Stream**:
   - For R&D specific UIs, keep it clean with **Main Text** color.
